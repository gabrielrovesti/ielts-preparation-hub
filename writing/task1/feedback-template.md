## Pre-Submission Checklist

Complete this **before** timing ends. Each unchecked box is a guaranteed score loss.

### Task Achievement (25%)

- [ ] All three bullet points addressed
- [ ] Each bullet point **extended** — not just mentioned, but developed with at least one follow-up detail or question
- [ ] No content invented beyond what the prompt specifies
- [ ] Opening line appropriate (e.g., not generic "I am writing to you...")
- [ ] Closing line appropriate to register

### Coherence & Cohesion (25%)

- [ ] Paragraphing: one bullet per paragraph (3-paragraph body, not a single block)
- [ ] Each paragraph has a clear single focus
- [ ] Transitions are natural, not mechanical ("Firstly / Secondly / Thirdly" is acceptable but not ideal)
- [ ] Reference and substitution used to avoid repetition (e.g., "the festival" → "it" / "the event")

### Register (critical for Task Achievement)

| Letter type | Required features |
|-------------|-------------------|
| Informal (friend/family) | Contractions, rhetorical questions, colloquial tone, personal remarks |
| Semi-formal (colleague, known manager) | Polite but not stiff; some contractions acceptable |
| Formal (unknown authority, company) | Full forms, professional vocabulary, passive structures if appropriate |

- [ ] Register consistent throughout (no mixing of formal opener with colloquial body)
- [ ] For informal: contractions used (`I'd`, `I've`, `it's`, `don't`)
- [ ] For informal: at least one personal/emotional remark per paragraph

### Lexical Resource (25%)

- [ ] No repetition of key words across paragraphs — substitution used
- [ ] No "very + basic adjective" constructions → use precise lexis
- [ ] Speculative options offered within questions where appropriate ("Is it X, Y, or Z?")
- [ ] No Italian interference patterns:
  - [ ] `demanded to` → `demanded that`
  - [ ] `provide equally` → `provide [X] equally`

### Grammatical Range & Accuracy (25%)

- [ ] Mix of simple and complex sentence structures
- [ ] Question forms correct (inversion where needed: "Could you tell me...?")
- [ ] No subject-verb agreement errors
- [ ] Tense consistency (present for current situation, past for completed events)

## Annotated Draft Section

After completing the letter, annotate the draft:

```
[TA+]  = strong task achievement move
[TA-]  = task achievement weakness / missed bullet extension
[LR+]  = strong lexical choice
[LR-]  = weak/repeated/Italian-interference lexis
[CC+]  = good cohesive device or paragraph structure
[CC-]  = cohesion gap or awkward transition
[GR-]  = grammatical error (note type: agreement / tense / form)
[REG]  = register mismatch
```
