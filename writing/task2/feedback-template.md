## Pre-Submission Checklist

### Task Response (25%)

- [ ] Position stated clearly in the introduction
- [ ] Position **consistent** across introduction, body, and conclusion (no drift)
- [ ] Both sides of the argument addressed (for discussion/opinion prompts)
- [ ] Every body paragraph is **directly relevant** to the prompt (no tangential content)
- [ ] Conclusion synthesises — does not merely repeat the introduction

### Coherence & Cohesion (25%)

- [ ] 4-paragraph structure: Introduction | Body 1 | Body 2 | Conclusion
- [ ] Each body paragraph follows: **claim → explanation → example/evidence**
- [ ] Cohesive devices used but not overused (avoid "Firstly, Secondly, Thirdly" as sole connectors)
- [ ] Paragraph-level progression: each paragraph advances the argument, not restates it
- [ ] Central argument visible from introduction through conclusion

### Lexical Resource (25%)

- [ ] No "very + basic adjective" constructions:

| Weak | Strong |
|------|--------|
| very expensive | costly / prohibitively expensive |
| very important | critical / fundamental / pivotal |
| very common | widespread / prevalent |
| very different | markedly distinct / substantially divergent |
| very big | substantial / considerable |
| very small | negligible / marginal |

- [ ] No direct Italian interference:
  - [ ] `demanded to` → `demanded that`
  - [ ] `provide equally` → `provide [X] equally`
  - [ ] `make a proposal` → `propose` (verb deflation)

- [ ] Academic vocabulary used naturally, not forced
- [ ] Key terms paraphrased across paragraphs (not repeated verbatim from prompt)

### Grammatical Range & Accuracy (25%)

- [ ] Mix of sentence types: simple, compound, complex, relative clauses
- [ ] Passive voice used where appropriate (not overused)
- [ ] Reported speech / nominalization where appropriate
- [ ] Conditional structures present (if any argument requires them)
- [ ] No tense inconsistency
- [ ] Subject-verb agreement correct throughout

## Essay Type Quick Reference

### "To what extent do you agree or disagree?"
- Take a clear directional stance (not full binary)
- Body 1: strongest argument for your position
- Body 2: constraint or nuance that qualifies your stance
- Conclusion: restate nuanced synthesis

### "Discuss both views and give your own opinion"
- Body 1: View A (with fairness — do not strawman)
- Body 2: View B + your qualified endorsement or rejection
- Conclusion: synthesise, state which view is better supported and why

### "Why does X happen? What can be done about it?"
- Body 1: causes (analytical, not a list)
- Body 2: solutions (specific, prioritised, tied to the causes)
- Avoid: solutions unrelated to the causes identified

### Two-part question (e.g., "Do you agree? What should governments do?")
- Each body paragraph addresses one part
- Never merge both parts into a single paragraph
