# Reading — Paraphrase Detection Drills

> Target skill: Band 8 sentence completion — meaning-first matching with full tolerance for lexical variation.  
> Band signal: Candidates who scan for surface-matching words ceiling at 6.5–7.

---

## Core Principle

In IELTS sentence completion, the **answer in the passage will never use the same words as the stem**. The relationship is always paraphrastic. Your job is to identify semantic equivalence across different surface forms.

**The five paraphrase mechanisms:**

| Mechanism | Example |
|-----------|---------|
| Synonym substitution | "excellent" → "high-quality" |
| Nominalisation collapse | "progression and development" → "progress" |
| Structural change | "it is not possible to do X" → "no X available" |
| Generalisation | "running, cycling, swimming" → "physical activity" |
| Specification | "the procedure" → "the three-stage approval process" |

## Drill Set 1 — Synonym Substitution

For each stem phrase, identify what synonym or paraphrase might appear in a passage.

**Exercise:** Write 2–3 alternative phrasings for each stem phrase. Check the key below.

| Stem phrase | Your alternatives | Key alternatives |
|-------------|------------------|------------------|
| "was enthusiastic about" | | keen on / showed great interest in / expressed strong support for |
| "led to a decline" | | resulted in a fall / caused a reduction / contributed to a drop |
| "difficult to understand" | | complex / opaque / hard to follow / not easily grasped |
| "required by law" | | legally obligatory / mandated / a legal requirement |
| "has improved significantly" | | increased substantially / risen considerably / advanced markedly |
| "the main reason" | | the primary cause / the chief factor / the dominant explanation |
| "made possible by" | | enabled by / facilitated by / owing to |
| "children" | | young people / minors / those under 18 / the younger generation |

## Drill Set 2 — Nominalisation and Structure Change

These are higher-difficulty paraphrases: the grammatical form changes, not just the vocabulary.

**Pattern:** Verb phrase in stem → noun phrase in passage (or vice versa)

| Stem (verb-based) | Passage form | Mechanism |
|-------------------|-------------|-----------|
| "the river flooded" | "the flooding of the river" | nominalisation |
| "they managed to survive" | "their survival" | nominalisation |
| "scientists discovered that..." | "the discovery of..." | nominalisation |
| "it is not possible to achieve this" | "no achievement of this is possible" / "this cannot be achieved" | negation + structure |
| "people increasingly prefer X" | "X has seen growing preference among consumers" | subject shift + nominalisation |

**Exercise:** For each item below, write what the passage might say (different structure, same meaning):

1. Stem: *"residents were dissatisfied with the service"*  
   Passage form: ___________________________

2. Stem: *"the policy aims to reduce costs"*  
   Passage form: ___________________________

3. Stem: *"experts disagree about the cause"*  
   Passage form: ___________________________

**Key:**
1. "complaints were received from residents" / "the service drew negative feedback" / "dissatisfaction among residents was reported"
2. "cost reduction is the stated objective of the policy" / "the policy targets expenditure cuts"
3. "the cause remains disputed among specialists" / "there is no consensus on the cause"

## Drill Set 3 — Semantic Field Traps

These are passages where a **surface match** is wrong and the **paraphrase** is correct.

**Example (from notes):**

| Stem: "excellent quality" | Passage: "high-quality teaching" | Correct: YES — adjective synonymy |
| Stem: "excellent quality" | Passage: "excellent results" | Correct: NO — "results" ≠ "quality" |

**Exercise:** For each item, determine whether the passage phrase matches the stem.

| Stem | Passage phrase | Match? | Reason |
|------|---------------|--------|--------|
| "progression and development" | "progress" | | |
| "progression and development" | "advanced programme" | | |
| "it is not possible to do" | "no facilities available" | | |
| "it is not possible to do" | "it is unnecessary to do" | | |
| "caused significant damage" | "led to substantial losses" | | |
| "caused significant damage" | "caused significant delays" | | |

**Key:**
- "progress" → YES (nominalisation collapse of "progression and development")
- "advanced programme" → NO (different semantic field)
- "no facilities available" → YES (structural change + synonym: "it is not possible" = "no X available")
- "it is unnecessary to do" → NO (unnecessary ≠ impossible)
- "led to substantial losses" → YES if context equates damage with financial/material loss; context-dependent
- "caused significant delays" → NO (delays ≠ damage in most contexts)

## Drill Set 4 — Distractor Training

Distractors are words in the passage that resemble the stem without satisfying it. Practice spotting them.

**Strategy:** When you find a word in the passage that matches the stem surface form, read the **full sentence in both the stem and passage** before committing. The match is at the **clause level**, not the word level.

**Exercise:** In each case, the passage contains a surface-level match that is a distractor. Identify why it fails.

1. **Stem:** *"The programme was successful in attracting international participants."*  
   **Passage A:** *"International interest in the programme grew steadily."*  
   **Passage B:** *"The programme attracted funding from international donors."*  
   **Correct passage:** ___  
   **Why the other fails:** ___

2. **Stem:** *"The new regulation placed limits on working hours."*  
   **Passage A:** *"Working hours in the sector had always been long."*  
   **Passage B:** *"The regulation capped the number of hours employees could work each week."*  
   **Correct passage:** ___  
   **Why the other fails:** ___

**Key:**
1. Passage A is correct ("growing interest" can be inferred to result from the programme being successful at attraction). Passage B fails because "donors" ≠ "participants".
2. Passage B is correct ("capped hours" = "placed limits on working hours"). Passage A fails because it describes a pre-existing situation, not the regulation's effect.

## Timed Drill Protocol

For authentic timed practice, use the following protocol with a Cambridge IELTS reading section:

1. Time: 60 minutes for the full section (Sections 1–3)
2. On sentence completion questions: **cover the answer options** and write your answer before looking (for non-MCQ types)
3. After completion: for every **wrong or uncertain answer**, identify which paraphrase mechanism you missed
4. Log the mechanism type (see table at top): most candidates have a consistent blind spot — isolating it is the key diagnostic
