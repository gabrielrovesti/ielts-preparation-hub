## Workflow

### Per practice session (writing)

1. Select a prompt from the relevant bank (alternate Task 1 and Task 2)
2. Complete the **positioning audit** (Task 2) or **register check** (Task 1) before writing
3. Write under timed conditions (20 min / 40 min)
4. Complete the **feedback template checklist** before reviewing any model answer
5. Generate or review a model answer; annotate your draft with the markup system
6. Record in the practice log; update status in the prompt bank

### Per practice session (reading)

1. Complete one full Cambridge reading section (60 min timed)
2. Self-mark against the answer key
3. For every incorrect sentence completion: identify which paraphrase mechanism was missed
4. Run the corresponding drill set from `paraphrase-drills.md`
