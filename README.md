## Workflow

### Per practice session (writing)

1. Select prompt from bank → complete positioning audit (T2) or register check (T1) before writing
2. Write under timed conditions (20 / 40 min)
3. Complete feedback template checklist before reviewing any model answer
4. Annotate draft; record in session tracker; update prompt bank status

### Per practice session (reading)

1. Full Cambridge section, timed (60 min)
2. Self-mark; categorise every error by paraphrase mechanism
3. Run the corresponding drill set

### Per practice session (listening)

1. Full Cambridge section, timed
2. Self-mark; categorise every error by type (see diagnostic table in listening guide)
3. Record in session tracker

### Per practice session (speaking)

1. Select cue card; 1-min prep; record 2-min response
2. Listen back; note top 3 issues; re-record once
3. Record in session tracker
